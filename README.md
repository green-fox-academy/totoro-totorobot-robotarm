# totoro-totorobot-robotarm

## Robot arm
### Short description
A 3 axis plotter robot arm, controlled from the STM32F746G-DISCO board.

The main goals are:
* can be used as regular robot arm
* controlled by joint and xyz coordinates
* can be used as a drawing machine
